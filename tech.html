Trinkit Engine & Software Development Kit

Trinkit is a cross-platform rhythm & pixel art game engine used by Boxsubmus to create 2D games with the simplicity of GameMaker and the flexibility of Unity.

Trinkit utilizes the lean on minimalism and simplicity of game development. Originally created for a fan-game, Trinkit is being developed primarily for use in rhythm and 2D retro games.

Trinkit was made to be all-in-one. Featuring a sprite editor, code editor, and runtime.
